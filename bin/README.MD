1. Công nghệ sử dụng
Backend: (Java)
- Spring Boot : bootstrapping project
- Spring MVC : mô  hình model-view-controller
- Spring Security
- Spring Session
- Spring Data  (Hibernate/JPA): sử dụng ORM (object relational mapping) để mô hình hóa Model

- Build tool:  Maven

Frontend: HTML, CSS và Javascript
- View Engine: JSP
- AJAX để gọi APIs 
Database: MySQL
